# SpeciesTracker
concept biological catalogue <br>
https://speciestracker2.azurewebsites.net/

# To Start: <br>
Download zip, open in code editor <br>
In terminal: nodemon

# Capabilities <br>
CRUD, Multiple tables <br>
image upload: npm i multer <br>
date & time autofill: npm/exif-js <br>
